# modern dark portfolio — personal landing page

my first full-scale multi-section landing page project. this isn't just a page, but a demonstration of modern layout skills, responsive design, and visual hierarchy.

→ **[view project live](https://taieth.github.io/taieth-web/)**

## 🎯 project goals
to create a minimalist, fast, and responsive interface that serves as a professional business card for a web developer. the main focus was on clean code and dark theme aesthetics.

## 🛠 tech stack
* **html5:** content structuring using sections (hero, services, about).
* **css3:** * using **flexbox** for flexible grids.
    * custom styles for service cards with smooth shadows and rounded corners.
    * global variables for easy color scheme management.
    * responsiveness (the site looks great on both desktop and mobile devices).
* **design:** focus on typography and whitespace.

## 📋 site structure
1. **hero section:** main screen with a clear call-to-action.
2. **services:** icon cards describing core competencies (design, development, promotion).
3. **about me:** section with personal story and development philosophy.
4. **navigation:** sticky menu for quick access to different sections.

## ✨ features
* fully **responsive layout**.
* buttons with smooth hover effects.
* high-quality svg icons for crisp visuals on any screen.
